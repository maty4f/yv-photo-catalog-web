בחר את סוג החומר שברצונך לקטלג:

📷

קטלוג תצלומים

זיהוי וקטלוג של תצלומים בודדים או אוספים שלמים, לפי הנהלים הרשמיים של מערכת ספיר.
  • אוסף + פריטים, או תצלום בודד
  • חזית + גב התצלום
  • זיהוי אישים, אובייקטים, סטודיו
  • פיענוח כיתובים רב-לשוני
  • קישור ישיר לתזאורוס הארכיון
🎞

קטלוג סרטים

זיהוי וקטלוג של סרטי ארכיון (אילמים ומדוברים), עם פירוט טיימליין ותיאור סצנה-אחר-סצנה.
  • סרטי סליל / וידאו / דיגיטלי
  • טיימליין מלא עם timecodes
  • תיאור סצנות, אישים, חפצים, סביבה
  • פיענוח כותרות ביניים וכתובות
  • זיהוי שפה, ז'אנר, איכות, צבעוניות
📄

קטלוג מסמכים

פיענוח מסמכים היסטוריים — מודפסים וכתב יד — בשפות שונות, עם תרגום לעברית ואנגלית.
  • גרמנית (כולל Fraktur), יידיש, פולנית, רוסית, צ'כית
  • כתב יד (Sütterlin, יידי) ומכונת כתיבה
  • תעתיק + תרגום עברית + אנגלית
  • זיהוי שולח, נמען, תאריך, מקום
  • קונטקסט היסטורי + נושאים מתזאורוס
🗂

קטלוג תיק שלם

קטלוג ברמת התיק — קריאת כל עמודי התיק (עשרות עד מאות) ויצירת רשומת ספיר אחת מסכמת.
  • העלאת תיקייה שלמה / מאות עמודים
  • קריאה דו-מנועית: Gemini קורא, Claude מסנתז
  • רשומת ספיר אחת לכל התיק
  • הקטנת סריקות חכמה — חוסך טוקנים
  • סימוני ודאות + דגלי אי-התאמה
💡 איך זה עובד: כל המערכות רצות כאן בדפדפן בלבד — ללא שרת, ללא התקנה. נדרש מפתח API אחד (Anthropic Claude או Google Gemini) — שניהם תומכים ב-OCR, ניתוח ויזואלי, ובניתוח טיימליין של סרטים. ה-API key נשמר רק במחשב שלך.
