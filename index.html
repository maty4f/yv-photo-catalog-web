1 תצלום Photograph
גרור לכאן תצלום, או לחץ לבחירת קובץ JPG / PNG / TIFF / HEIC / WEBP
2 מידע מוקדם Provided context — אופציונלי
הכל קורה בדפדפן שלך — שום דבר לא נשלח לאף שרת. הקובץ ייקרא מקומית, יחולץ הטקסט, וזה ימולא בשדה למעלה.
3 יצירת פרומפט ל-Claude Generate Claude prompt

לוחץ על הכפתור — נוצר פרומפט עם כל הכללים של יד ושם + הקונטקסט שלך, ומועתק ללוח. אז פותח Claude.ai, מצרף את התצלום ומדביק.

4 קבלת הפלט מ-Claude Paste Claude's HTML output

Claude יחזיר HTML מלא לפי התבנית של ספיר. הדבק את כל הקובץ כאן — אנשמור אותו בשם `photo_<תאריך>.html` ותוכל לפתוח ולהשתמש בו.

💡 על האבטחה: כל העיבוד מתבצע בדפדפן שלך בלבד. תצלומים, קונטקסט, וקבצים שאתה מעלה — לא נשלחים לאף שרת. הספרייה היחידה שנטענת מהאינטרנט היא קוד-JS לחילוץ PDF/Word/Excel (מ-jsDelivr CDN), שאינו רואה את הקבצים שלך. אם הצלחה לטעון את העמוד הזה במחשב העבודה — זה אומר שהאינטרנט הבסיסי פתוח, וגם הספריות יטענו.
