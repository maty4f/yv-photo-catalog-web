📍 ⭐ Hybrid (מומלץ — דיוק היסטורי): Gemini עושה זיהוי ראשוני — OCR נאמן בשפת המקור (חינמי, איכותי) → Claude עושה זיהוי משני — בונה את שדות קטלוג ספיר, מתרגם לעברית אקדמית, מאמת היסטורית (סוג מסמך, צדדים, תאריכים, מקומות). שניהם מסמנים ודאות dual-axis V/H: V=ויזואלי (מה קריא במסמך), H=היסטורי (מה תואם תיעוד). דורש שני מפתחות. עלות ~$0.002/דף. ראה reference/confidence_dual_axis.md.
📍 🆓 Gemini בלבד (חינם): OCR + ניתוח במעבר אחד, בדרך כלל בתוך ה-free tier. ללא שכבת אימות היסטורי בעברית אקדמית. מסלול חינמי חלופי (מומלץ למסמכים מורכבים): yv doc catalog <file> --pipeline-free — Gemini API (free tier) + claude CLI שלך = pipeline אמיתי, $0.
📍 למסמכים מודפסים: שני הספקים עובדים מצוין. Gemini Flash הכי זול.
📍 לכתב יד מאתגר: Hybrid או Transkribus — דיוק גבוה יותר ושכבת אימות היסטורי.
📍 🇪🇺 Transkribus: סטנדרט הזהב לכתב יד אירופאי. 500 קרדיטים/חודש חינם (~50-100 עמ׳). דרוש חשבון ב-app.transkribus.org. אם API חסום (CORS) — בחר "הזנה ידנית" ותדביק את ה-OCR.
📍 שפות נתמכות: גרמנית (כולל Fraktur/Sütterlin), יידיש, פולנית, רוסית, צ'כית, סלובקית, הונגרית, אוקראינית, ליטאית, לטבית, רומנית, יוונית, עברית, אנגלית, צרפתית, איטלקית.
