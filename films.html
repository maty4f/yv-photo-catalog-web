📊 מקרא סימוני ודאות / Confidence markers

כל זיהוי מדורג בשני צירים — סוג הראיה (אות) ורמת הוודאות (צבע וסימן).

סוג הראיה / Evidence type
Vויזואלי — מה שנראה בסרט עצמו Visual — what the film shows
Hהיסטורי — התאמה למקורות ולמידע מתועד Historical — match to documented sources
רמת ודאות / Confidence level
✓ודאי — עדות חד-משמעית Confirmed — decisive evidence
~סביר — מתיישב עם הרמזים Probable — consistent with cues
?משוער — דמיון או הסקה בלבד Tentative — resemblance/inference only

דוגמאות: V✓H✓ כיתוב-ביניים קריא בסרט שתואם מקור מתועד · V?H✓ פנים מטושטשות אך הקונטקסט ההיסטורי ודאי.

📍 Pipeline (מומלץ — דיוק היסטורי): Gemini עושה זיהוי ראשוני — רואה את הסרט ומפיק טיוטה של סצנות, כותרות ביניים, אישים → Claude עושה זיהוי משני — מקבל את הטיוטה + 20 פריימים ומאמת היסטורית (מתקן מקומות, תאריכים, מדים, טרמינולוגיית יד-ושם). שניהם מסמנים ודאות dual-axis V/H: V=ויזואלי (מה רואים בסרט), H=היסטורי (מה תואם תיעוד). דורש שני מפתחות. עלות מצטברת אבל דיוק גבוה. ראה reference/confidence_dual_axis.md.
📍 🆓 Gemini בלבד (חינם) — מהיר וזול לסקירה כללית, ללא שכבת אימות היסטורי. בדרך כלל בתוך ה-free tier של Google:
· עד ~18MB: inline (בקשה אחת)
· 18MB–2GB: Files API — העלאה אוטומטית עם progress bar
· מעל 2GB: יש לדחוס/לחתוך מראש (HandBrake)
· מסלול חינמי חלופי: yv film extract <file> מקומית → yv film catalog <file> מדפיס prompt → הדבק ב-Claude Code (שיש לך) → גם זה $0.
