# yv-photo-catalog — web edition

גרסת web סטטית של מערכת קטלוג התצלומים. רצה כולה בדפדפן, ללא שרת, ללא התקנה.

## למה גרסה זו?

מיועדת למחשבי עבודה עם הגבלות אבטחה שלא מאפשרות:
- התקנת Python / Flask / spam packages
- הרצת סקריפטים מקומיים
- חיבור לשרתים פרטיים (כמו mf-sr.com שחסום)

הגרסה הזו עובדת **בכל דפדפן מודרני**, מ-domain של `github.io` (כמעט אף פעם לא חסום).

## מה היא עושה

1. **קלט תצלום** — drag-and-drop בדפדפן
2. **קלט קונטקסט** — חופשי או חילוץ אוטומטי מקובץ:
   - **PDF** (PDF.js)
   - **Word .docx** (Mammoth.js)
   - **Excel .xlsx** (SheetJS)
   - **CSV / TSV** (JS מקומי)
   - **TXT / MD / JSON / RTF**
3. **יצירת פרומפט** עם כל הכללים של יד ושם — מועתק ללוח
4. **פתיחת Claude.ai** — מדביק שם, מצרף תצלום, מקבל HTML
5. **שמירת ה-HTML** — מדביק חזרה, מוריד כקובץ מקומי

## פרטיות ואבטחה

- **שום דבר לא נשלח לאף שרת** — כל העיבוד בדפדפן
- ספריות ה-JS נטענות מ-jsDelivr CDN (קוד בלבד, לא רואה את הקבצים שלך)
- אחרי טעינה ראשונה, הדפדפן יקדים אותן — אפשרי שיעבוד גם offline
- התצלום עצמו נשלח ל-Claude.ai רק כשאתה בעצמך מצרף אותו שם

## ספריות חיצוניות

- [PDF.js](https://mozilla.github.io/pdf.js/) — חילוץ טקסט מ-PDF
- [Mammoth.js](https://github.com/mwilliamson/mammoth.js) — חילוץ מ-Word
- [SheetJS](https://sheetjs.com/) — חילוץ מ-Excel

נטענות ב-CDN. אם הוספה תהיה חסומה — שמור את הדף offline (Cmd+S, "save webpage complete") והוא ימשיך לעבוד עם הספריות שכבר בקשו.
